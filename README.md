# Tienda de Mods de Metro Exodus

## Plan de pruebas:
- **1.- Login / Registro de usuarios.
- **2.- CRUD de Mods (Subir, editar o borrar un mod de la base de datos).
- **3.- Buscador y filtros (ej. buscar "filtros de máscara de gas").
- **4.- Carrito de compras / Proceso de Checkout.
- **5.- Sistema de valoraciones (darle 5 estrellas a un mod).
- **6.- Verificar que funcione el tema del envio de mensaje.
